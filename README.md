## 📖 Overview

Zen Chess is a premium, multi-page frontend web application built as a comprehensive practice hub for chess players. It features a fully playable AI practice board, a high-speed vision training mini-game, and a state-persistent user dashboard.

## ✨ Key Features

- **Playable Engine:** Integrated with `chessboard.js` and `chess.js` for legal move validation and AI responses.
- **Vision Drill Mini-Game:** Custom DOM manipulation and event listeners to create a 30-second speed drill for board coordination.
- **Multi-Page Routing:** Seamless navigation between Practice, Vision, and Dashboard interfaces.
- **Session Management:** Utilizes HTML5 `localStorage` to simulate secure login, registration, and stat tracking across multiple pages.
- **Security Routing:** Active page guards redirect unauthenticated users away from the dashboard.
- **Modern UI/UX:** Custom dark-slate aesthetic, fully responsive Bootstrap layout, and CSS animations.

## 🛠️ Tech Stack

- **Core:** HTML5, CSS3, Vanilla JavaScript
- **Styling:** Bootstrap 5.3, Custom CSS Variables
- **Libraries:** jQuery, Chessboard.js, Chess.js
